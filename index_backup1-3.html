属性面板换算

物理攻击
(敏捷 × 0.725 + 武器攻击 + 模组攻击) × (1 + 攻击加成%) = 物理攻击
暴击计算
[暴击 / (暴击 + 4458.1)] × 100% + 5% = 总暴击%
幸运计算
[幸运 / (幸运 + 4458.1)] × 100% + 5% = 总幸运%
急速与攻速
[急速 / (急速 + 4458.1)] × 100% + 基础急速% = 总急速%; 总急速% × 1.6 = 攻速%
精通计算
[精通 / (精通 + 4458.1)] × 100% + 6% = 总精通%
全能计算
[全能 / (全能 + 2501.1)] × 100% = 总全能%
生命上限
耐力 × 7.8 + 生命上限基础值 = 总生命上限
元素加成
[元素强度 / (元素强度 + 4458.1)] × 100% = 元素加成%
全元素加成
全元素强度 × 2.2% = 全元素加成%
